# LookTheWorld——一个专门为拿不到手机的苦逼学生制作的软件

## 使用方法
1. 在[这里](https://github.com/Dennis114514/LookTheWorld/releases "这里")下载APK安装包
2. 从https://github.com/Dennis114514/LookTheWorld.git 克隆源码，然后编译
```bash
git clone https://github.com/Dennis114514/LookTheWorld.git
#Windows环境
.\gradlew build
#其他系统
gradlew build
```
## 功能特点
1. 极致兼容，最低支持Android4.1  
注：安卓4.2以下未经测试，可能有bug
2. 信息多样，目前有来自chinanews.com.cn的新闻，来自IT之家的科技动态，来自HoyoLAB的原神资讯
3. 手表兼容：为手表显示进行了优化
4. 多源稳定：对于使用RSShub的源，除了官方源外，还有7个镜像源以增强稳定性
